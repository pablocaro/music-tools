Music Tools

A collection of interactive music experiments and prototypes

  • Piano Prototype

    Original radial piano prototype with circular layout and touch support
  • Piano 2

    Second iteration of the radial piano with improved dark theme
  • Piano Refactored

    Refactored version with cleaner code structure and enhanced features including pinch-to-zoom and octave markers
  • Piano Experimental

    Experimental branch for testing new features and interactions
  • Piano Experimental (Refactored)

    Modular version with separated HTML, CSS, and JavaScript for better maintainability
  • Sound Testing Lab

    Comprehensive Tone.js testing environment for experimenting with synthesizers, envelopes, and filters to find the perfect sound
  • Simple Sound Lab

    Super simple testing environment with a single circle to play notes
  • Piano Visual Only

    Visual-only version of the radial piano interface without audio
  • Tone.js Learning Lab

    Interactive learning platform teaching Tone.js from first principles with hands-on lessons and experiments
